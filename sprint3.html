Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'. (But not those ones).

The term control flow refers to the order in which the instructions in your program are executed. Loops are a really fundamental concept in programming. A loop is essentially something that will continue to repeat until a certain condition is met (i.e. until a certain condition becomes true).

Real Life example:

Leaving the house with my daughter I learn this lesson after so many times when we went somewhere, I always end up missing something from her nappy bag (either spare clothes in case something happens, nappies, cream etc..). I got so tired of this situation constantly happening to me, that I created a control flow (checklist) of things I have to do before I leave the. This is the list:

  1. Find Matti’s bag
  2. Add nappies
  3. Check if there is Nappy rash cream
  4. Add spare clothes
  5. Fill bottle with water
  6. Put the bottle in the bag
  7. Find her Dolly
  8. Put Dolly in the bag

So, if for any strange reason some of this element from this list is not met or does not appear inside of the bag, there is when Mr Loop appear because I know I can’t leave the house without one of this element inside of the bag.

Describe what the DOM is and an example of how you might interact with it.

The Document Object Model is a platform- and language-neutral interface that will allow programs and scripts to dynamically access and update the content, structure and style of documents. The document can be further processed and the results of that processing can be incorporated back into the presented page.

Explain the difference between accessing data from arrays and objects.

The difference between accessing data from arrays and objects is the quality and the kind of information we can get when we call this tool. Array is a list of information not specifically displayed on the screen e.g. the way how our phone saves information about our contacts list:

You can access the information by pointing to a specific item in the array e.g. friends[0] will give you contact information for John.
